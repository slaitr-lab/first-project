## Выделение кода

Чтобы выделить текст как код, поместить его в тройные кавычки - тильды.

```Python
def convert(number):
    return (format(number, 'b'), format(number, 'o'), format(number, 'X'))
print(convert(15))
```

Это лишь некоторые функции markdown.
